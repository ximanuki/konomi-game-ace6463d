# 🎮 このみちゃんゲームアプリ - ゲームデザインドキュメント (GDD)

**対象ユーザー:** このみちゃん（6歳）  
**デバイス:** iPhone SE2 (375×667 CSS px)  
**コンセプト:** 安全で楽しく、達成感のある学びと遊びの体験

---

## 🎯 全体設計方針

### ユーザー体験の柱
1. **シンプルな操作**: タップ、ドラッグのみ
2. **即座のフィードバック**: アクションに対して音と視覚で応答
3. **ポジティブな体験**: 失敗より成功を強調、励ましの言葉
4. **適切なセッション時間**: 1回3-10分、集中力を考慮
5. **安全性**: 広告なし、外部リンクなし、オフライン完結

### フィードバック設計
- **褒める**: 「すごい！」「よくできたね！」「かんぺき！」
- **励ます**: 「もうちょっと！」「がんばって！」
- **サウンド**: タップ音、正解音、クリア音、BGM（オフ可能）
- **視覚効果**: キラキラ、弾むアニメーション、色の変化

## 1. 🧩 ジグソーパズル

### ゲームの目的
画像を分割されたピースから元の絵に復元する

### コアメカニクス
1. 画像を3×3（9ピース）に分割
2. ピースをランダムに配置
3. タップ＆ドラッグでピースを移動
4. 正しい位置にスナップ（磁石のように吸着）
5. 全ピース完成で勝利

### 難易度設計
- **かんたん**: 3×3（9ピース）- デフォルト
- **ふつう**: 4×3（12ピース）- 将来実装
- **むずかしい**: 4×4（16ピース）- 将来実装

### 勝利条件
全ピースが正しい位置に配置される

### フィードバック
- **ピース配置**: 正しい位置に近づくと光る
- **正しい配置**: 「パチッ」とスナップ音
- **完成**: 「🎉 やったね！かんせい！」+ 紙吹雪アニメーション + クリア音

### セッション時間
5-8分（6歳の集中力に適切）

### 画像選択
- 動物（🐱🐶🐼）
- 食べ物（🍓🍰🍩）
- 乗り物（🚗✈️🚂）
- 自然（🌸🌈⭐）

※ 各カテゴリ3-5枚、合計15-20枚を用意

## 2. 🎴 絵合わせ（神経衰弱）

### ゲームの目的
裏返しのカードから同じ絵のペアを見つける

### コアメカニクス
1. 3×4グリッド（12枚）= 6ペア
2. カードは裏返しで配置
3. タップで1枚目をめくる
4. タップで2枚目をめくる
5. 同じ絵 → カードが消える + 正解音
6. 違う絵 → 1秒後に自動で裏返る
7. 全ペア揃えたら勝利

### 難易度設計
- **かんたん**: 3×2（6枚/3ペア）- 導入用
- **ふつう**: 3×4（12枚/6ペア）- デフォルト
- **むずかしい**: 4×4（16枚/8ペア）- 将来実装

### 勝利条件
全ペアを揃える

### フィードバック
- **カードめくり**: フリップアニメーション + めくり音
- **正解**: 「ピンポン♪」+ カードが弾む → 消える
- **不正解**: 「あれれ？」（音なし、静かに裏返る）
- **完成**: 「🎉 ぜんぶみつけたね！」+ 星が降る

### セッション時間
3-7分

### 絵柄
- 動物の顔: 🐱🐶🐰🐼🐸🐷
- フルーツ: 🍎🍊🍇🍓🍌🍉
- 乗り物: 🚗🚕🚙🚌🚎🏎️
- 季節: 🌸🌻🍁❄️🌈⭐

## 3. 🎨 おえかき [新規]

### ゲームの目的
自由にCanvas上に絵を描いて創造性を発揮

### コアメカニクス
1. Canvasに指で自由に描画
2. 色選択（8色パレット）
3. ペン太さ選択（細・中・太）
4. 消しゴム機能
5. 全消去ボタン
6. スタンプ機能（絵文字を配置）

### ツール
- **描画色**: ❤️🧡💛💚💙💜🖤🤍（8色）
- **太さ**: 細(3px) / 中(8px) / 太(15px)
- **消しゴム**: 太さ固定(20px)
- **スタンプ**: ⭐❤️😊🌸🐱🍎（6種類）

### 操作
- タップで色選択
- ドラッグで描画
- 消しゴムモードで消す
- スタンプモードでタップ配置
- 「ぜんぶけす」ボタンで確認ダイアログ → クリア

### フィードバック
- **色選択**: タップ音 + 選択枠表示
- **描画中**: 滑らかな描画、遅延なし
- **スタンプ配置**: ポンッ音 + 弾むアニメーション
- **クリア**: 「ほんとうにけしていい？」確認

### セッション時間
自由（制限なし、好きなだけ）

### 保存機能
- ローカルストレージに自動保存（最新1枚）
- アプリ再起動時に復元

## 4. 🌈 いろあわせ [新規]

### ゲームの目的
色の名前とカードをマッチングして色を学ぶ

### コアメカニクス
1. 画面上部に色名を表示（例: 「あか」）
2. 下部に4色のカードを表示
3. 正しい色をタップ
4. 正解 → 次の問題へ
5. 不正解 → ブルッと振動、再挑戦
6. 10問正解でクリア

### 色の種類
- あか（赤）
- あお（青）
- きいろ（黄）
- みどり（緑）
- ピンク
- オレンジ
- むらさき（紫）
- ちゃいろ（茶）
- くろ（黒）
- しろ（白）

### 難易度設計
- **かんたん**: 4色（赤・青・黄・緑）
- **ふつう**: 6色（+ピンク・オレンジ）
- **むずかしい**: 8色（+紫・茶）

### 勝利条件
10問連続正解

### フィードバック
- **正解**: 「ピンポーン♪」+ カードが弾む + キラキラ
- **不正解**: ブルッ（触覚フィードバック）+ カードが小さく揺れる
- **クリア**: 「🌈 いろマスター！」+ 虹のアニメーション

### セッション時間
2-4分

### 固有SE

- オレンジ: ラ

## 5. 🎵 リズムゲーム [新規]

### ゲームの目的
流れてくるマークをタイミングよくタップしてリズム感を養う

### コアメカニクス
1. 画面上から下に向かってマーク（⭐❤️🌸）が流れる
2. 画面下部の判定ラインでタップ
3. タイミング判定: Perfect / Good / Miss
4. スコアを加算
5. 曲が終わったらクリア

### 曲
- **きらきらぼし** - シンプルなメロディ（30秒）
- **ちょうちょ** - ゆったりテンポ（30秒）
- **かえるのうた** - 楽しいリズム（30秒）

※ Web Audio APIで生成、著作権フリー

### 判定
- **Perfect**: ±50ms → +10点 + 「パーフェクト！」
- **Good**: ±150ms → +5点 + 「グッド！」
- **Miss**: それ以外 → 0点 + 「...」

### 難易度設計
- **かんたん**: BPM 60、マーク間隔1秒
- **ふつう**: BPM 90、マーク間隔0.75秒
- **むずかしい**: BPM 120、マーク間隔0.5秒

### 勝利条件
曲を最後まで演奏（ミスしてもクリア扱い）

### フィードバック
- **タップ**: 判定に応じた音（高音/中音/低音）
- **Perfect**: キラキラエフェクト
- **Good**: 小さな光
- **Miss**: 無反応（静か）
- **クリア**: スコア表示 + 「おつかれさま！」

### セッション時間
1曲30秒 × 3曲 = 約2-5分

## 6. 🌸 おはなそだて [新規]

### ゲームの目的
種を植えて水やりをし、花を育てて収穫する育成ゲーム

### コアメカニクス
1. **種を選ぶ**: アンロック済みの種から選択
2. **プランターに植える**: 最大3つ同時育成可能
3. **水やり**: 1日1回タップで水やり（アニメーション）
4. **成長**: 3日で開花（水やりでスピードアップ）
5. **収穫**: 開花したらタップで収穫
6. **図鑑に記録**: 収穫した花が図鑑に追加

### 成長ステージ
- **1日目**: 🌱 芽（種を植えた直後）
- **2日目**: 🌿 葉（水やり1回後）
- **3日目**: 🌸 花（水やり2回後、開花）
- **収穫後**: 空のプランター

### 花の種類
- **初期**:
  - 🌸 さくら（ピンク）
  - 🌷 チューリップ（赤）
- **アンロック**:
  - 🌻 ひまわり（黄）→ 実績「ガーデナー」
  - 🌹 バラ（赤）→ 実績「おはなマスター」
  - 💐 ブーケ（カラフル）→ 実績「おはなはかせ」

### 水やり忘れ対策
- **枯れない**: 水やりを忘れても花は枯れない（子供向け配慮）
- **成長が遅くなる**: 水やりしないと成長が止まる
- **通知**: 「おはなが おみずをまってるよ！」（バッジ表示）

### 勝利条件
全種類の花を開花させる

### フィードバック
- **種植え**: 「ポンッ」音 + 種が土に埋まるアニメーション
- **水やり**: ジョウロから水が出るアニメーション + 「ジャー」音
- **成長**: 「ポンッ」と成長アニメーション
- **開花**: 「🎉 おはながさいたよ！」+ キラキラエフェクト
- **収穫**: 「やったね！」+ 花が手に入るアニメーション

### セッション時間
- 1回の操作: 30秒-1分（水やりのみ）
- 毎日のルーティン: 楽しみに待つ要素

### 図鑑機能
収穫した花を記録：
- 花の名前
- 収穫日
- 合計収穫数
- 「あと○しゅるい！」進捗表示

### 特別な仕掛け
- **レア花**: 低確率で色違いが咲く（🌺🌼等）
- **季節限定**: 特定の日に特別な花（未実装、将来拡張用）
