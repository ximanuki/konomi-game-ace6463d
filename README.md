# 🎮 このみちゃんのゲーム

6歳の女の子「このみちゃん」のための PWA ゲームアプリ

## 🌟 特徴

- **完全オフライン対応**: インターネット不要
- **PWA**: ホーム画面に追加して本物のアプリのように使える
- **iPhone SE2 最適化**: 完璧なタッチ操作とレイアウト
- **音声フィードバック**: Web Audio API で楽しいサウンド
- **実績システム**: 30以上の実績をアンロック
- **進捗保存**: LocalStorage で自動保存

## 🎮 ゲーム一覧

### 1. 🧩 ジグソーパズル
画像を分割されたピースから復元するパズルゲーム
- 3×3ピース
- ドラッグ＆ドロップ操作
- ベストタイム記録

### 2. 🎴 絵合わせ（神経衰弱）
同じ絵のペアを見つける記憶ゲーム
- 3×4グリッド（12枚）
- カードフリップアニメーション
- 最小手数記録

### 3. 🎨 おえかき
自由に絵を描いて保存できる創作ゲーム
- 8色パレット
- 3段階のペンサイズ
- スタンプ機能
- ギャラリー（最大10作品保存）

### 4. 🌈 いろあわせ
色の名前を学ぶ教育ゲーム
- 4択クイズ形式
- 各色に固有の音階
- 10問クリアでゲーム完了

### 5. 🎵 リズムゲーム
音楽に合わせてタップするリズムゲーム
- 3曲収録（きらきらぼし、ちょうちょ、かえるのうた）
- Perfect/Good/Miss 判定
- スコア記録

### 6. 🌸 おはなそだて
種を植えて花を育てる育成ゲーム
- 5種類の花
- 毎日の水やり
- 収穫・図鑑システム

## 🏆 実績・アンロックシステム

### 実績
- **30以上の実績**: 各ゲームに複数の実績
- **称号**: マスター、はかせ、でんせつ
- **連続プレイ記録**: 7日連続で特別な実績

### アンロック要素
- **パズル画像**: 初期3枚→最大10枚
- **絵合わせテーマ**: 初期2種→最大6種
- **おえかきスタンプ**: 初期6個→最大26個
- **花の種**: 初期2種→最大5種

## 🎨 マイクラ・牧場物語要素

### 創作の自由（マイクラ要素）
- おえかきギャラリー（10作品保存）
- 自由な表現

### コレクション（牧場物語要素）
- 30+実績のコンプリート
- 花の図鑑
- スタンプコレクション

### 育成要素
- おはなそだて（毎日水やり）
- 3日で開花

### 毎日遊びたくなる
- 連続プレイ記録
- 花の成長
- デイリー実績

## 🚀 使い方

### オンライン版
https://ximanuki.github.io/konomi-game-ace6463d/

### ホーム画面に追加（推奨）
1. Safari でアクセス
2. 共有ボタン → 「ホーム画面に追加」
3. アイコンをタップして起動

## 📱 対応環境

- **推奨**: iPhone SE2 (iOS 15+)
- **対応**: iOS Safari, Chrome (モバイル)
- **画面サイズ**: 375×667 CSS px 最適化
- **オフライン**: 完全対応

## 🛠️ 技術スタック

- **HTML5 + CSS3**: セマンティックマークアップ
- **Vanilla JavaScript**: ライブラリ不使用
- **Web Audio API**: サウンドシステム
- **Canvas API**: 描画・パズル
- **LocalStorage API**: 進捗保存
- **Service Worker**: オフライン対応
- **PWA Manifest**: アプリ化
